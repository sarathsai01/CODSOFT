# Spam SMS Detection

## Overview
Spam message detection is a common application of natural language processing (NLP) and machine learning. The objective of this project is to build a model that can automatically classify SMS messages as spam or legitimate (ham).

## Dataset
The dataset contains labeled SMS messages categorized as spam or ham.

Dataset Source:
https://www.kaggle.com/datasets/uciml/sms-spam-collection-dataset

Target Labels:
- ham → legitimate message
- spam → unwanted or promotional message

## Methodology
The following steps were used to build the spam detection system:

1. Data loading and preprocessing
2. Text vectorization using TF-IDF
3. Splitting the dataset into training and testing sets
4. Training a Multinomial Naive Bayes classifier
5. Evaluating the model using accuracy and classification metrics

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Natural Language Processing

## Model Performance
The model achieved approximately **96–97% accuracy** in detecting spam messages.

## Conclusion
This project shows how machine learning and text processing techniques can be applied to automatically detect spam messages and improve communication security.
